API Documentation

Base URL

https://785sk4gpbh.execute-api.us-east-1.amazonaws.com/dev

Authentication

Include your API key in the X-API-Key header:

X-API-Key: your-api-key-here

Available Endpoints

Public Endpoints (No API Key Required)

  • GET /public/message - Get a public message
  • POST /auth/login - User authentication
  • POST /developer/register - Developer registration

Protected Endpoints (Require Authentication)

  • GET /auth/message - Get authenticated message
  • GET /items - List all items
  • POST /items - Create new item
  • GET /items/{id} - Get specific item
  • PUT /items/{id} - Update item
  • DELETE /items/{id} - Delete item

Rate Limits

  • Rate Limit: 100 requests per second
  • Burst Limit: 200 requests
  • Daily Quota: 10,000 requests per day

Example Usage

JavaScript/Fetch

fetch('https://785sk4gpbh.execute-api.us-east-1.amazonaws.com/dev/items', { method: 'GET', headers: { 'X-API-Key': 'your-api-key-here', 'Authorization': 'Bearer your-jwt-token-here' } }) .then(response => response.json()) .then(data => console.log(data));

cURL

curl -H "X-API-Key: your-api-key-here" \ -H "Authorization: Bearer your-jwt-token-here" \ https://785sk4gpbh.execute-api.us-east-1.amazonaws.com/dev/items
